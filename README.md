# LLM Evaluation Framework & Expert-annotated Motivational Interviewing (MI) Dataset

This repository contains the following files:

* `train_dataset.xlxs`: The original training dataset file.
* `test_dataset.xlxs`: The original test dataset file.
* `train_annotated.xlxs`: The MI IDs and annotated original training dataset. 
* `test_annotated.xlxs`: The MI IDs and annotated original test dataset.
* `ChatGPT_MI_sessions.xlxs`: 97 augmented MI sessions adhering to MITI.
* `ChatGPT_annotated_sample.xlxs`: 15 expert-annotated MI sessions sample (representative of the training dataset).
* `Gemini_MI_sessions.xlxs`: 97 augmented MI sessions adhering to MITI.
* `Gemini_annotated_sample.xlxs`: 15 expert-annotated MI sessions sample (representative of the training dataset).
* `DeepSeek_MI_sessions.xlxs`: 97 augmented MI sessions adhering to MITI.
* `DeepSeek_annotated_sample.xlxs`: 15 expert-annotated MI sessions sample (representative of the training dataset).
